# Easy Effects Presets: ThinkPad E14 & Sony WH-1000XM5

This repository contains two [Easy Effects](https://github.com/wwmm/easyeffects) presets tuned for different listening scenarios.

## 1. ThinkPad E14 Gen 7 AMD (`thinkpad.json`)
A profile designed to improve the sound of the built-in laptop speakers. 

It is built on top of the excellent preset originally created by [@sebastian-de](https://github.com/sebastian-de), but heavily modified.
* **Key modules:** Multiband Compressor, Loudness, Crystalizer, Reverb, and a final Limiter.
* **Goal:** Improve dynamic range, clarify high frequencies, and prevent digital saturation.

## 2. Sony WH-1000XM5 (`WH-1000XM5.json`)
A high-fidelity corrective profile tailored for these closed-back headphones.

It is based on an [AutoEq](https://autoeq.app/) profile, with some personal tweaks to fit my own listening preferences.
* **Key modules:** 10-band Parametric Equalizer (PEQ) and a subtle Reverb.
* **Goal:** Neutralize the factory bass boost using an accurate parametric curve (aligned to the Harman target) and restore instrument separation, adding slight reverberation to improve the spatial image.

Feel free to fork and adjust them to your liking.
